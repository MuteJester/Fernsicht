How it works

Three steps. One line in your code.

01Wrap

Add one line.

Wrap any loop, long task, or shell command. The CLI works with no code change; the Python and R SDKs offer a one-import path inside your code.

02Run

Run your code.

A shareable link appears in your terminal — printed once. Send it in Slack, paste it into a ticket, put it on a fridge iPad. The receiver doesn't need to install anything.

03Watch

Watch from anywhere.

Open the link on any device. Progress streams directly from your machine to theirs over WebRTC — peer-to-peer, end-to-end, with no intermediate server storing your data.

Free forever · no accounts

Fernsicht is free. Help us keep it that way.

There's no paywall and no account wall; there never will be. A small signalling server brokers the initial handshake between sender and viewer — after that, your data stays peer-to-peer. Support keeps the lights on and the rendezvous open for everyone.
